# 3.3.3 Error Suggestion

- Level: AA
- Principle: 3 Understandable
- Guideline: 3.3 Input Assistance
- Understanding: https://www.w3.org/WAI/WCAG22/Understanding/error-suggestion.html
- How to meet: https://www.w3.org/WAI/WCAG22/quickref/#error-suggestion

## Requirement

If an input error is automatically detected and suggestions for correction are known, then the suggestions are provided to the user, unless it would jeopardize the security or purpose of the content.

## Definitions used

### input error

information provided by the user that is not accepted

Note

This includes:

Information that is required by the web page but omitted by the user
Information that is provided by the user but that falls outside the required data format or values
