# 2.4.7 Focus Visible

- Level: AA
- Principle: 2 Operable
- Guideline: 2.4 Navigable
- Understanding: https://www.w3.org/WAI/WCAG22/Understanding/focus-visible.html
- How to meet: https://www.w3.org/WAI/WCAG22/quickref/#focus-visible

## Requirement

Any keyboard operable user interface has a mode of operation where the keyboard focus indicator is visible.

## Definitions used

### focus indicator

New

pixels that are changed to visually indicate when a user interface component is in a focused state
